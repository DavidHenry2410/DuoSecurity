# Get-DuoAccounts

## SYNOPSIS
List Duo child accounts

## SYNTAX

```
Get-DuoAccounts [<CommonParameters>]
```

## DESCRIPTION
Calls the Accounts API to retrieve Duo Accounts under parent

## EXAMPLES

### EXAMPLE 1
```
Get-DuoAccounts
```
