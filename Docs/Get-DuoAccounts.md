# Get-DuoAccounts

## SYNOPSIS
Retrieve Accounts

## SYNTAX

```
Get-DuoAccounts [<CommonParameters>]
```

## DESCRIPTION
Returns a list of child accounts.

## EXAMPLES

### EXAMPLE 1
```
Get-DuoAccounts
```
