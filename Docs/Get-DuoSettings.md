# Get-DuoSettings

## SYNOPSIS
Retrieve Settings

## SYNTAX

```
Get-DuoSettings [<CommonParameters>]
```

## DESCRIPTION
Returns global Duo settings.
These settings can also be viewed and set in the Duo Admin Panel.
Requires "Grant settings" API permission.

## EXAMPLES

### EXAMPLE 1
```
Get-DuoSettings
```
