# Get-DuoIntegrations

## SYNOPSIS
Retrieve Integrations

## SYNTAX

```
Get-DuoIntegrations [[-IntegrationKey] <String>] [<CommonParameters>]
```

## DESCRIPTION
Returns a single integration or a paged list of integrations.
Requires "Grant read resource" API permission.

## EXAMPLES

### EXAMPLE 1
```
Get-DuoIntegrations
```
