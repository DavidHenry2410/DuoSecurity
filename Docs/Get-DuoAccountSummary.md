# Get-DuoAccountSummary

## SYNOPSIS
Retrieve Summary

## SYNTAX

```
Get-DuoAccountSummary [<CommonParameters>]
```

## DESCRIPTION
Returns a summary of account utilization information.
Requires "Grant read information" API permission.

## EXAMPLES

### EXAMPLE 1
```
Get-DuoAccountSummary
```
