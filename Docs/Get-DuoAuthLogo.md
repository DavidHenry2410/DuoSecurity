# Get-DuoAuthLogo

## SYNOPSIS
Duo Auth Logo

## SYNTAX

```
Get-DuoAuthLogo [-FilePath] <String> [<CommonParameters>]
```

## DESCRIPTION
The /logo endpoint provides a programmatic way to retrieve your stored logo.

## EXAMPLES

### EXAMPLE 1
```
Get-DuoAuthLogo -FilePath ./logo.png
```
